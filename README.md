# Biblioteca Online
[![NPM](https://img.shields.io/npm/l/react)](https://github.com/GalvanGabe/biblioteca_online/blob/main/LICENSE) 

# About the project

A web system that adds, queries, edits and deletes books from a database (MySQL), simulating a small online library. Project made through the classes of Professor Matheus Battisti’s NodeJS course (from the Hora de Codar channel) on the Udemy platform, with some layout changes and data that are sent to the database(MySQL).
# Sobre o projeto

Um sistema web que adiciona, consulta, edita e exclui os livros de um banco de dados(MySQL), simulando uma pequena biblioteca online. Projeto feito através das aulas do curso de NodeJS do professor Matheus Battisti (do canal Hora de Codar) na plataforma Udemy, com algumas alterações de layout e dados que são enviados para o banco de dados(MySQL).
